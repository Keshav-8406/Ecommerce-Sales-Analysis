 # 🛒 Ecommerce Sales Analysis

## 📌 About Project
This project is based on analyzing sales data of an e-commerce company using Python and Pandas.

Since real data was not available, a synthetic dataset of 200 transactions was created.

The main goal is to understand:
👉 Sales performance  
👉 Top regions  
👉 Best salespersons  
👉 Category-wise sales  

---

## 📊 Dataset Details

The dataset includes:

- 🆔 transaction_id  
- 🌍 region (North, South, East, West)  
- 📦 category  
- 👤 salesperson  
- 💰 sales_amount  
- 🧑 customer_id  

---

## 🧠 Tasks Performed

### 🔹 Task 1: Basic Analysis
✔ Total sales by region  
✔ Transactions per category  
✔ Average sales per salesperson  

---

### 🔹 Task 2: Advanced Analysis
✔ Region + Category sales  
✔ Salesperson performance (sum, mean, count)  
✔ Top category using highest revenue  

---

### 🔹 Task 3: Custom Analysis
✔ Created custom function (sales_range)  
✔ Region-wise detailed report  
✔ Final summary report  

---

## 📈 Key Insights

✨ Some regions perform better than others  
✨ Certain categories generate more revenue  
✨ Salesperson performance varies
